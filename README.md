# examples-rl-lidar
Examples of how various RL algorithms perform in a Gymnasium environment mimicking LIDAR readings.
* simple_grid: taken from Gymnasium example code, simple implementation of Q learning in 5x5 grid environment
## Directory structure
simple_grid:        
   │   agent.py
   │   doTest.py
   │   environment.py
   │   main.py
   │   plot.py
   │   q_values.npy
   │   test.py
   │   train.py
## Requirements

`pip install numpy torch gymnasium`
